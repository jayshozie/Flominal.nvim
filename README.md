# Flominal.nvim

This plugin provides a somewhat customizable floating terminal.
Almost entirely stolen from TJ, I promise I'll work on it.

# Setup

## Lazy.nvim

```
'jayshozie/Flominal.nvim'
config = function()
    require('Flominal')
end
```
